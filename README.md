# HireRemote.ph Platform

HireRemote.ph is a remote job marketplace connecting Filipino professionals with companies hiring globally.

Our platform uses automation and AI to help companies discover talent in the Philippines while giving skilled professionals access to international opportunities.

Website
https://hireremote.ph

## Platform Components

- Remote job discovery engine
- AI-powered job search
- Salary transparency database
- Employer hiring tools
- Global remote work marketplace

## Mission

HireRemote.ph exists to expand global opportunities for the Philippine workforce and help companies build distributed teams faster.
